## System Overview

PromptPal is a React Native + Expo Router app. The UI is organized with file-based routing under `src/app/`, feature state is grouped under `src/features/`, and shared app services live in `src/lib/`. The app integrates Clerk authentication and Convex for backend data, with Google Gemini used through Convex mutations. Safe mode (`EXPO_PUBLIC_SAFE_MODE=1`) shows a branded `BootModeScreen` for crash isolation; otherwise the root lazy-loads `NormalRoot` (GestureHandler + Clerk + Convex + ErrorBoundary).

## Navigation Guide

**To add a new API endpoint**: `convex/mutations.ts` or `convex/queries.ts`; update `convex/schema.ts` if needed.

**To add a new onboarding step**: `src/features/onboarding/store.ts` (steps array), add screen in `screens/`, wire in `OnboardingFlow.tsx`.

**To add a new component**: `src/components/ui/` for shared primitives; `src/features/<feature>/components/` for feature-specific.

**To modify auth**: `src/lib/clerk.tsx`, `src/lib/auth.ts`, `convex/auth.config.ts`.

**To modify scoring**: `src/lib/scoring/copyScoringCore.ts`, `promptQuality.ts`; Convex `ai.ts` for server-side evaluation.
